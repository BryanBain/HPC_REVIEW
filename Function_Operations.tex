\section{Operations with Functions: Domain}

When adding, subtracting, multiplying functions, the {\color{red}\textbf{domain}} of this is the overlap (on a number line) of the domains of each function. \newline 

In other words, if you graph the domain of each function on a number line, the overlap is the part of the number line those shaded areas have in common. \newline 

You can also find the domain of the sum, difference, product, or quotient \emph{before} you simplify the expression. \newline 

\emph{Note:} Division is the same with the added condition that the denominator is not allowed to equal 0. 

\begin{example}
Given $f(x) = \frac{3}{x^2-4x+3}$, $g(x) = \sqrt{5x+10}$, and $h(x) = 2x+2$, state the domain of each.
\begin{multicols}{2}
\begin{enumerate}[(a)]
    \item $(f + g)(x)$
    \item $(g \cdot h)(x)$
\end{enumerate}
\end{multicols}
\end{example}

\begin{solution}

\begin{itemize}
    \item Domain of $f(x)$:
    \begin{align*}
        x^2 - 4x + 3 &\neq 0 \\
        (x - 1)(x - 3) &\neq 0 \\
        x \neq 1 &\text{ or } x \neq 3
    \end{align*}
    \item Domain of $g(x)$:
    \begin{align*}
        5x + 10 &\geq 0 \\
        x &\geq -2
    \end{align*}
    \item Domain of $h(x)$: \quad $\mathbb{R}$
\end{itemize}

\vspace{12pt}

(a) $(f + g)(x) = \frac{3}{x^2-4x+3} + \sqrt{5x+10}$   \newline 

Combining the domains of $f(x)$ and $g(x)$ from above on a number line:

\begin{center}
\begin{tikzpicture}
\draw [<->] (0,0) -- (6,0);
\draw (1.5, 0.15) -- (1.5, -0.15) node [below] {$-2$};
\draw (3, 0.15) -- (3, -0.15) node [below] {1};
\draw (4.5, 0.15) -- (4.5, -0.15) node [below] {3};
\draw [->, thick, red] (1.5, -1) -- (6, -1) node [right] {$g(x)$};
\draw [red, fill=red] (1.5,-1) circle [radius = 2.5pt];
\draw [color = blue, thick] (3,1) circle [radius = 2.5pt];
\draw [color = blue, thick] (4.5,1) circle [radius = 2.5pt];
\draw [->, thick, blue, shorten <= 2.5pt] (3,1) -- (0,1);
\draw [thick, blue, shorten <= 2.5pt, shorten >= 2.5pt] (3,1) -- (4.5,1);
\draw [->, thick, blue, shorten <= 2.5pt] (4.5, 1) -- (6, 1) node [right] {$f(x)$};
\end{tikzpicture}
\end{center}

Looking at where the intervals of $f(x)$ and $g(x)$ overlap gives us the following. \newline 

\begin{center}
\begin{tikzpicture}
\draw [<->] (0,0) -- (6,0);
\draw (1.5, 0.15) -- (1.5, -0.15) node [below] {$-2$};
\draw (3, 0.15) -- (3, -0.15) node [below] {1};
\draw (4.5, 0.15) -- (4.5, -0.15) node [below] {3};
\draw [violet, fill=violet] (1.5,0) circle [radius = 2.5pt];
\draw [thick, violet] (3,0) circle [radius = 2.5pt];
\draw [thick, violet] (4.5,0) circle [radius = 2.5pt];
\draw [ultra thick, violet, shorten >= 2.5pt] (1.5,0) -- (3,0);
\draw [ultra thick, violet, shorten <= 2.5pt, shorten >= 2.5pt] (3,0) -- (4.5,0);
\draw [->, ultra thick, violet, shorten <= 2.5pt] (4.5,0) -- (6,0);
\end{tikzpicture}
\end{center}

In interval notation, our solution is $[-2, 1) \cup (1, 3) \cup (3, \infty)$ \newline 

(b) $(g \cdot h)(x) = \left(\sqrt{5x+10}\right)\left(2x+2\right)$ \newline 

Combining the domains of $g(x)$ and $h(x)$ on a number line:

\begin{center}
\begin{tikzpicture}
\draw [<->] (0,0) -- (6,0);
\draw (1.5, 0.15) -- (1.5, -0.15) node [below] {$-2$};
\draw (3, 0.15) -- (3, -0.15) node [below] {1};
\draw (4.5, 0.15) -- (4.5, -0.15) node [below] {3};
\draw [->, thick, red] (1.5, -1) -- (6, -1) node [right] {$g(x)$};
\draw [red, fill=red] (1.5,-1) circle [radius = 2.5pt];
\draw [->, thick, blue, shorten <= 2.5pt] (3,1) -- (0,1);
\draw [thick, blue, shorten <= 2.5pt, shorten >= 2.5pt] (3,1) -- (4.5,1);
\draw [<->, thick, blue] (0, 1) -- (6, 1) node [right] {$h(x)$};
\end{tikzpicture}
\end{center}

Looking at where the intervals of $g(x)$ and $h(x)$ overlap gives us the following.

\begin{center}
\begin{tikzpicture}
\draw [<->] (0,0) -- (6,0);
\draw (1.5, 0.15) -- (1.5, -0.15) node [below] {$-2$};
\draw (3, 0.15) -- (3, -0.15) node [below] {1};
\draw (4.5, 0.15) -- (4.5, -0.15) node [below] {3};
\draw [violet, fill=violet] (1.5,0) circle [radius = 2.5pt];
\draw [->, ultra thick, violet] (1.5,0) -- (6,0);
\end{tikzpicture}
\end{center}

In interval notation, our solution is $[-2, \infty)$
\end{solution}
