\chapter{Angles and Radian Measure}

Sketch each of the following. Then find a coterminal between 0 and $360^\circ$ (or 0 and $2\pi$ radians) for each.

\begin{enumerate}
	\item $-\frac{3\pi}{4}$
	\item $900^\circ$
	\item $\frac{27\pi}{10}$
	\item $-125^\circ$
\end{enumerate}	\setcounter{Review}{\value{enumi}}
